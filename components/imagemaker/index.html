openBalena Image Maker

Build a ready-to-flash installable image, pre-configured for a fleet.

Flash the downloaded .img.gz with balenaEtcher. For generic-amd64 the .img.gz is a flasher image — boot the target from it and it installs balenaOS to the device's disk, then auto-registers into the fleet.
Jetson Orin/Xavier: flash via USB recovery mode with jetson-flash (not Etcher). Jetson Nano: SD image, Etcher works.
